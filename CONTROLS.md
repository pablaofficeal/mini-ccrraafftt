# Управление камерой в 3D пространстве

## Клавиши управления:

### Движение:
- **W** - движение вперед
- **S** - движение назад  
- **A** - движение влево
- **D** - движение вправо
- **Пробел** - подъем вверх
- **C** - спуск вниз

### Мышь:
- **Движение мыши** - поворот камеры
- Мышь автоматически захватывается при запуске

### Система:
- **ESC** - выход из приложения

## Особенности:

- Камера движется в направлении, куда смотрит
- Угол наклона ограничен для предотвращения переворота
- Плавное движение с постоянной скоростью
- 3D пространство с большой плоской поверхностью (20x20 единиц)
- 5 красных кубов в разных позициях для тестирования
- Реалистичное освещение с тенями и бликами
- Правильная трансформация нормалей для корректного освещения

## Технические детали:

- Позиция камеры: (0, 2, 5) в начале
- Начальный наклон: -20 градусов (смотрит немного вниз)
- Скорость движения: 0.1 единицы за кадр
- Чувствительность мыши: 0.1 градуса за пиксель
- Частота обновления: ~60 FPS
